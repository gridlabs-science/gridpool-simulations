## Paper-Ready Interpretation

This run confirms that the public GridPool nodes expose state, peer, event, and relay telemetry, and that state agreement can be checked across independently hosted nodes. In this sample, reachable primary nodes can be compared directly and external operator nodes can be flagged if they diverge. The sample is too small and too operationally dependent to prove global latency behavior or UDP effectiveness; it is a field sanity check for observability and state health.
